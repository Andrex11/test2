Introducción a CSS y Selectores básicos

¿Que es CSS?

CSS es un lenguaje de diseño gráfico que permite definir y crear la presentación de un documento estructurado escrito en un lenguaje de marcado. Es muy usado para establecer el diseño visual de los documentos web e interfaces de usuario escritas en HTML.

¿Que significa CSS?

Las siglas CSS (Cascading Style Sheets) significan «Hojas de estilo en cascada» y parten de un concepto simple pero muy potente: aplicar estilos (colores, formas, márgenes, etc...) a uno o varios documentos (generalmente documentos HTML, páginas webs) de forma automática y masiva.

¿Por qué se usa CSS?

La idea de CSS es la de utilizar el concepto de separación de presentación y contenido. Este concepto se basa en que, como programadores, lo ideal es separar claramente el código que escribimos. ¿Por qué? Porque con el tiempo, esto hará que el código sea más fácil de modificar y mantener.

Estructura de CSS

Los documentos CSS son archivos de texto (igual que los documentos HTML), sólo que en este caso tienen extensión .css en lugar de .html. En el código de un documento .css se escriben una serie de órdenes y el cliente (navegador) las interpreta y aplica a los documentos HTML.

Relación de el HTML con CSS

En primer lugar, recuerda que debemos tener el documento .css enlazado desde nuestro documento .html, preferiblemente desde una etiqueta <link rel="stylesheet">.

Documento CSS

Los documentos CSS se forman por cientos de reglas de este estilo, con diferentes selectores, propiedades y valores. Habitualmente, cada bloque declarado suele incluir múltiples reglas.


Debemos tener en cuenta que, a medida que escribimos código CSS, este se va haciendo más grande y más difícil de controlar y mantener. Para intentar evitar perder el control del mismo, hay que seguir una serie de buenas prácticas al escribir código, que aunque no son necesarias para que funcione en el navegador, los desarrolladores consideramos obligatorias para que sea más fácil de leer:


  • Escribe una regla por línea. Será mucho más fácil de leer y modificar.

  • Usa la indentación. Tras escribir el carácter { al empezar un bloque de reglas, las propiedades se separan hacia la derecha. Esto hace que sea más fácil de leer y se considera una buena práctica de desarrollo y de programación en general.

  • El último punto y coma de un bloque de reglas es opcional. Sin embargo, se suele aconsejar escribirlo para mantener una coherencia y evitar problemas o descuidos al modificar posteriormente el código.

Estos consejos mejoran sustancialmente la legibilidad del código. Se consideran prácticas de cumplimiento obligatorio para ayudar a entender más rápidamente el código ajeno (o incluso el nuestro) sin necesidad de invertir mucho tiempo.

Selectores

Los selectores en CSS son patrones que se utilizan para seleccionar y aplicar estilos a elementos HTML específicos en una página web. Los selectores permiten a los diseñadores y desarrolladores web definir reglas de estilo que se aplicarán a elementos específicos en función de su estructura, atributos o clases. A continuación, se presentan algunos de los selectores más comunes en CSS:

  1. Selector de tipo (Element Selector):
    Se utiliza para seleccionar todos los elementos de un tipo específico en una página. Por ejemplo, para seleccionar todos los párrafos en un documento, puedes usar p como selector.
    2. CSS

    p {
    color: blue;      /* Cambia el color de los parrafos a azul */
    }

  2. Selector de clase (Class Selector):
    Permite seleccionar elementos que tienen un atributo class específico. Se denota con un punto antes del nombre de la clase.
    3. CSS

    .destacado {
    font-weight: bold;     /* Hace que el texto de estos elementos esté en negrita */
    }

  3. Selector de ID (ID Selector):
    Se utiliza para seleccionar un elemento específico con un atributo id. Se denota con un signo de almohadilla (#) seguido del nombre del ID.
    4. CSS

    #encabezado {
    background-color: #ccc;      /*Establece un fondo gris claro a este elemento */
    }

  4. Selector de descendencia (Descendant Selector):
    Permite seleccionar elementos que son descendientes de otros elementos. Se utilizan espacios para separar los selectores.
    5. CSS

    div p {
    font-style: italic;     /* Cambia el estilo del texto a cursiva en los parrafos dentro de "div" */
    }

  5. Selector de hijo directo (Child Selector):
    Similar al selector de descendencia, pero selecciona solo los elementos que son hijos directos de otro elemento. Se utiliza el signo mayor que (>) para separar los selectores.
    6. CSS

    ul > li {
    list-style-type: square;      /* Cambia el estilo de la lista a viñetas cuadradas */
    }

  6. Selector de atributo (Attribute Selector):
    Se utiliza para seleccionar elementos basados en sus atributos. Puedes seleccionar elementos con un atributo específico y, en algunos casos, con un valor específico del atributo.
    7. CSS

    input [type="text"] {
    border: 1px solid #ccc;      /* Agrega un borde gris alrededor de los elementos de entrada de tipo 'text' */
    }

  7. Selector universal (Universal Selector):
    Selecciona todos los elementos en la página y se denota con un asterisco (*).
    8. CSS

    * {
    margin: 0;      /* Elimina el margen de todos los elementos */
    padding: 0;      /* Elimina el relleno de todos los elementos */
    }

Propiedades

Las propiedades CSS (Cascading Style Sheets) son instrucciones que se utilizan para controlar la presentación y el diseño de elementos HTML en una página web. Estas propiedades permiten definir características como el color, el tamaño, la posición, el espaciado y muchas otras características visuales de los elementos HTML. Ejemplos

  1. Propiedad Color:
    Define el color de el texto
    2. CSS

    p {
    color: red;
    }

  2. Propiedad "font-size":
    Establece el tamaño de la fuente
    3. CSS

    h1 {
    font-size: 24px;
    }

  3. Propiedad background-color:
    Define el color de fondo de un elemento
    4. CSS

    div {
    background-color: red;
    }

  4. Propiedad margin:
    Controla el espacio externo alrededor de un elemento
    5. CSS

    img {
    margin: 10px;
    }

  5. Propiedad padding:
    Controla el espacio interno dentro de un elemento
    6. CSS

    div {
    padding: 15px;
    }

  6. Propiedad border:
    Define un borde alrededor de un elemento
    7. CSS

    button {
    border: 1px solid   #ffffff;
    }

  7. Propiedad text-aling:
    Alinea el texto de un elemento
    8. CSS

    p {
    text-aling: center;
    }

  8. Propiedades width y height:
    Controlan el ancho y la altura de un elemento
    9. CSS

    img {
    width: 200px;
    heigth: 150px;
    }

  9. Propiedad font-family:
    Establece la fuente de texto utilizada.
    10. CSS

    body {
    font-family: Arial, sans-serif;
    }

  10. Propiedad display:
    Contorla el tipo de visulaización de un elemento, como bloque o en linea
    11. CSS

    span {
    display: inline;
    }
